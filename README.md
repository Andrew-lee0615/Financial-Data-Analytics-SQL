# 📊 Financial Data Analytics: SQL Repository

This repository serves as a **Technical Asset Library** for financial data manipulation and risk analysis. It documents the transition from Excel-based modeling to scalable SQL queries for institutional-grade data environments.

## 🛠 Project Scope
* **Data Retrieval:** High-performance filtering of asset classes and tickers.
* **Risk Metrics:** Implementation of SQL-based aggregation for portfolio risk assessment.
* **Data Integration:** Relational mapping of disparate financial datasets (Price, Fundamentals, Macro).

## 🗂 Library Structure
* **01_Extraction_Logic:** Basic filtering, sorting, and distinct data retrieval.
* **02_Aggregation_Analysis:** Portfolio statistics, summing exposures, and counting assets.
* **03_Relational_Mapping:** Multi-table joins for comprehensive risk dashboards (Upcoming).
* **04_Advanced_Analytics:** Window functions for time-series financial data (Upcoming).
